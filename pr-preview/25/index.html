Grid-Sight — Dynamic Sliders

Continuous, pixel-precise sliders attached to numeric table axes. Live linear / bilinear interpolation, four-cell highlight, optional formula readout, cross-table sync, URL persistence, heatmap marker, and threshold fade — no runtime dependencies, full offline support.
